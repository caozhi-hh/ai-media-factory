# 自己调视频 —— 大白话指南

> 想改配音 / 背景图 / 文字布局,**不用找 Claude**,改 `config/` 里的文件,再重新跑 `python main.py "书名"` 就行。
> 改完**一定要存盘**,然后重新跑才生效。

---

## 一、改声音 / 语速 / 音量 → `config/voice.yaml`

打开 `config/voice.yaml`,改这几行:

```yaml
model: zh-CN-YunxiNeural   # 换声音(下面有列表)
rate: "+0%"                 # 语速:+10% 快,-10% 慢
volume: "+0%"               # 音量:+20% 大声,-20% 小声
```

**换声音**:把 `model:` 后面换成下面任一个(去掉注释 `#`):

| 声音 | model 值 |
|---|---|
| 温柔年轻男(默认) | `zh-CN-YunxiNeural` |
| 阳刚男 | `zh-CN-YunjianNeural` |
| 温暖中年男 | `zh-CN-YunyeNeural` |
| 温柔女(晓晓) | `zh-CN-XiaoxiaoNeural` |
| 活泼年轻女 | `zh-CN-XiaoyiNeural` |

**语速举例**:嫌慢就 `rate: "+15%"`,嫌快就 `rate: "-10%"`。

## 二、改背景图 → `config/video.yaml` 的 `background:`

三种方式选一个:

### 方式 A:用网上风景图(默认),换一张
```yaml
background:
  source: picsum
  seed: 活着        # ← 改成别的词(如"夕阳""麦田""海边"),换另一张图
```

### 方式 B:用你自己找的图(推荐,最贴合)
1. 把你想用的图(竖屏 1080×1920 最好)保存到 `assets/background.jpg`(覆盖原文件)
2. 改配置:
```yaml
background:
  source: local
  local_path: assets/background.jpg
```

### 字看不清?(背景太亮)
```yaml
  dim: 0.45    # ← 调大,0.6 / 0.7 背景更暗,字更清楚
```

---

## 三、改文字 / 布局 → `config/video.yaml` 的 `text:`

```yaml
text:
  font_size: 54        # 字大小,嫌小就 72、80
  color: white         # 字颜色:white / black / yellow / cyan
  position: center     # 位置:center 居中 / top 靠上 / bottom 靠下
  line_spacing: 28     # 行距,字挤一起就调大
```
