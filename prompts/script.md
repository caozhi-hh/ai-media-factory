## 写作要求

### 1. 开头(必须金句,前两句定生死)
- **第一句必须是金句、警句、震撼人心的陈述**,例如"孤独是一种永恒的颜色"、"我们终其一生,不过是在重复父辈的命运"。
- **禁止**用"嗨朋友们"、"家人们"、"今天给大家介绍"、"你是否也…"这种口水化开场。
- 开头要在 3 秒内让人停下划走——抛出一个洞察、一个反问、或者一句直接击中情绪的话。

### 2. 中段(浓缩全书精华,不要讲故事梗概)
- 用 **80-100 字** 讲出全书最核心的一个主题、一个意象、一个贯穿始终的隐喻。
- **不要"讲发生了什么"**(那是剧透),而是讲"这本书在讲什么"——它揭示了什么样的命运、人性、或者时代的真相。
- 可以引用书中著名的比喻、象征(金句、意象),但不要堆砌。

### 3. 结尾(共鸣,留白,不要喊口号)
- 结尾一句要让人听完之后停顿 2 秒,而不是"你还在等什么"这种营销感。
- 用一句总结性的话、或者把开头金句再换角度呼应一次。
- **不要"建议所有人去读"、"强烈推荐"、"点赞收藏"、"关注我"**——这些话只会让人反感。

### 4. 整体风格
- 长度:**严格 180-220 字**(读完约 28-32 秒, 目标 30 秒视频). 少于 150 字直接重写.
- 节奏:口语化但有文学质感, 短句多, 长句少
- 用第一人称("你"称呼读者, "我们"称呼作者)
- **不要 emoji, 不要感叹号连用, 不要 hashtag, 不要"全书精华""经典语录""必读"这种营销词**
- **每句之间用 \\n 换行隔开** (重要! 字幕按 \\n 分段, 不换行会糊成一句)

### 4.5 钩子分类 (开头第一句必须从下面 3 类选一类)
**根据书的气质选钩子类型, 详见 hook_styles.md:**

- **A. 治愈型** (文学/成长/疗愈): "这本书 + 治好了/替我说出/懂了我 + [长期情绪]"
  - 例: "这本书, 治好了我二十年的不满足。"
- **B. 反常识型** (社科/历史/科普): "[普遍认同的观点] + 其实是错的/恰恰相反"
  - 例: "你以为努力就能成功? 这本书说, 顺序反了。"
- **C. 数字冲击型** (商业/传记/实用): "[数字] + [反差/对比] + [震撼结论]"
  - 例: "他 40 岁放弃一切, 只为了一句话。"

**判定指引:**
- 文学小说/成长励志 -> A 治愈型
- 社科/历史/心理/科普 -> B 反常识型
- 商业/财经/传记/实用 -> C 数字冲击型

### 4.6 Few-shot 示例 (照这个长度和结构写)

**示例 1 (文学/治愈型钩子, 《月亮与六便士》):**
```
这本书, 治好了我二十年的不满足。
"我总得谨记, 他是个不错的商人。"
"满地都是六便士, 他却抬头看见了月亮。"
"我不想谋生, 我想生活。"
"走着走着, 就到了一辈子想待的地方。"
如果有一天你也想抬头, 别怕, 月亮一直都在。
```

**示例 2 (财经/数字冲击型钩子, 《富爸爸穷爸爸》):**
```
他 40 岁还在打工, 50 岁成了富翁, 区别只是一句话。
穷人为了钱工作, 富人让钱为自己工作。
学校教你怎么考试, 却从不教你怎么赚钱。
"资产是把钱放进你口袋的东西, 负债是把钱从你口袋拿走的东西。"
大多数人一辈子都在为别人的梦想买单。
你呢, 你是在为谁工作?
```


### 6. 简介区文案(发布时单独写, 不混进口播稿)
**核心原则: 简介 ≠ 正文复读。**
- 口播稿(本 prompt 输出的) = 视频里念的、引用书中原话的金句
- 简介区文案 = 你(博主)读完书后的**第一人称感悟**, 跟读者对话, 不是书中引用
- 简介不要带引号、不要复制书里原句、不要重复口播稿的内容
- 简介的读者是**没看过视频的人**(从主页点进来先看简介), 要给"为什么要看"的理由

**简介写作模板(6 句对仗, 每句 15-25 字):**
1. 读了 N 遍这本书才敢推荐给你。(降低姿态)
2. 前 N-1 遍觉得 XX, 最后一遍才懂, (给出阅读转折)
3. 他不是 XX, (反常识铺垫)
4. 他只是 XX。(颠覆性结论)
5. 如果你也 XX, (绑定读者经历)
6. 这本书会替你把它说出来。(引导点开)

**简介禁忌:**
- 不要"强烈推荐""必读""全网最细"等营销词
- 不要 hashtag(那是话题区的事)
- 不要复制口播稿的引号金句到简介

### 5. 输出格式
**只输出文案正文,纯文本,不含任何提示词/标题/标签**。
